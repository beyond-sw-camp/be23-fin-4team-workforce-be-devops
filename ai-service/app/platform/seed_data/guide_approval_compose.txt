[결재] 결재 작성

■ 어떤 작업?
회사에 결재를 요청합니다 (휴가, 출장, 사직서, 공문 등 모든 결재).

■ 어디서?
사이드바 → 전자결재 → 결재 생성

■ 작성 절차
1. 결재 화면 진입
2. "결재 생성" 클릭
3. 양식 선택 - 사용 가능한 양식 목록:
   - 휴가신청서 (연차/반차/경조 등)
   - 연장근무신청 (사전/사후, 연장/야간/휴일)
   - 출퇴근시간 변경 신청서
   - 휴직 신청서 (출산/육아/장기병가 등)
   - 사직서
   - 근로계약서
   - 수당 변경 신청
   - 국내출장신청
   - 해외출장신청
   - 업무기안
   - 업무보고서
   - 공문
4. 입력 항목 작성
5. 결재 라인 확인
6. 제출 또는 임시저장

■ 자주 묻는 질문
- 결재 어떻게 작성해? → 전자결재 > 결재 생성
- 결재 어떻게 올려? → 전자결재 > 결재 생성
- 결재 어디서 시작해? → 전자결재 > 결재 생성
- 어떤 양식이 있어? → 휴가/출장/사직서/근로계약/공문 등 (회사별 다름)
- 임시저장 가능? → 가능, "내 기안 문서 > 임시 저장함"에서 확인
- 결재 양식 만들고 싶어 → 관리자만 가능 (전자결재 > 결재 관리자)
- 한 번 제출한 결재 수정하려면? → 회수 후 재작성
- 결재 처음 어떻게 시작해? → 전자결재 메뉴 > 결재 생성
