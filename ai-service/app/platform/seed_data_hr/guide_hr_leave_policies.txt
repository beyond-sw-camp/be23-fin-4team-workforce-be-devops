[인사관리] 연차 정책 관리

■ 어떤 작업?
회사 연차 정책을 등록·수정·삭제합니다.
연차 발생 기준(회계연도 vs 입사일), 기본 부여 일수, 근속별 추가 부여, 촉진제도, 이월, 미사용 연차 수당 지급 등을 종합 관리합니다.
근로기준법 제60조(연차)·제61조(연차 사용 촉진) 기준에 맞춰 회사 정책을 정의합니다.

■ 어디서?
사이드바 → 휴무 관리 → 연차 정책 관리

■ 화면 구성
[목록 표]
- 발생 기준: 회계연도 / 입사일 라벨
- 기본 부여(일): 1년차 부여 일수 (예: 17일)
- 촉진제도: 사용 / 미사용 (사용 시 1차/2차 알림 일수 표시)
- 이월: 허용 (N일) / 불허
- 미사용 수당: 지급 / 미지급
- 액션: 편집·삭제 아이콘

[우상단 버튼]
- "연차 통보 미응답자 관리": 촉진제도 통보 후 직원 응답을 추적
- "정책 추가": 신규 연차 정책 등록 모달

[정책 등록 모달]

1. 발생
- 발생 기준: 회계연도 (매년 1/1 일괄 부여) / 입사일 (직원별 입사일 기준 부여)
- 기본 부여 일수 (1년차): 법정 최소 15일 (1년 이상 근속자)
- 추가 부여 주기: 근로기준법 매 2년마다 (디폴트)
- 추가 일수: 근로기준법 1일씩 (디폴트)
- 최대 한도: 근로기준법 25일 (회사가 더 후하게 가능, 더 낮게는 불가)
- 근속별 연차 미리보기: 1년차/3년차/5년차/.../21년차 자동 계산 표시

2. 촉진제도 (선택사항, 근로기준법 제61조)
- ON 시 미사용 연차 사용을 사전 안내, 운영 시 미사용 수당 지급 의무 면제 가능
- 1차 촉진 알림: 만료일 N일 전 (예: 180일 전)
- 2차 촉진 알림: 만료일 N일 전 (예: 90일 전)

3. 이월 (선택사항)
- ON 시 회계연도 종료 후 미사용 연차를 다음 해로 넘김. 매년 1/1 자동 처리
- 최대 이월 일수
- 이월 동의서 사용 여부

4. 미사용 연차 수당 (선택사항)
- ON 시 만료된 미사용 연차를 다음 1월 급여 항목으로 자동 추가
- 계산식: 통상시급(기본급 ÷ 209) × 8시간 × 미사용 일수

[정책 요약 패널 - 시나리오]
- 현재 토글 상태에 따른 동작 시나리오를 실시간 표시
- 예: 미사용 연차가 그대로 소멸 / 다음 해 이월 / 수당으로 지급

■ 시스템 동작 (코드 기반)
- 근속 N년 시점 연차 = 기본 부여 + floor((N-1) / 추가 주기) × 추가 일수, 최대 한도로 cap
- 발생 기준 FISCAL: 매년 1/1 일괄 부여 (입사일은 별도 처리)
- 발생 기준 HIRE_DATE: 입사 기념일에 부여
- 촉진제도 미운영 시 미사용 연차 수당 지급 의무
- 이월·수당 모두 OFF면 미사용 연차는 그대로 소멸 (보상 없음)

■ 자주 묻는 질문 (위치/방법 안내)
- 연차 정책 어디서 만들어? → 휴무 관리 > 연차 정책 관리 > "정책 추가"
- 연차 정책 어떻게 등록? → 연차 정책 관리 화면 > 우상단 "정책 추가"
- 연차 발생 기준 어디서 설정? → 연차 정책 관리 > "정책 추가" > 발생 기준
- 회계연도 기준 어디서 설정? → 연차 정책 관리 > 정책 추가/수정 > 발생 기준 "회계연도"
- 입사일 기준 어디서 설정? → 연차 정책 관리 > 정책 추가/수정 > 발생 기준 "입사일"
- 연차 일수 어디서 변경? → 연차 정책 관리 > 정책 편집 > 기본 부여 일수
- 기본 연차 어디서 설정? → 연차 정책 관리 > "정책 추가" > 기본 부여 일수
- 연차 상한 어디서 설정? → 연차 정책 관리 > "정책 추가" > 최대 한도
- 근속 가산 어디서 설정? → 연차 정책 관리 > "정책 추가" > 추가 부여 주기/추가 일수
- 촉진제도 어디서 설정? → 연차 정책 관리 > "정책 추가" > 촉진제도 사용 토글
- 연차 촉진 어디서? → 휴무 관리 > 연차 정책 관리
- 1차 촉진 어디서 설정? → 연차 정책 관리 > 정책 편집 > 촉진제도 ON > 1차 촉진 알림 일수
- 이월 어디서 설정? → 연차 정책 관리 > "정책 추가" > 이월 허용 토글
- 연차 이월 어디서? → 휴무 관리 > 연차 정책 관리 > 이월 허용 ON
- 미사용 수당 어디서 설정? → 연차 정책 관리 > "정책 추가" > 미사용 연차 수당 지급 토글
- 연차 수당 어디서? → 휴무 관리 > 연차 정책 관리 > 미사용 연차 수당
- 연차 통보 미응답자 어디서? → 휴무 관리 > 연차 정책 관리 > 우상단 "연차 통보 미응답자 관리"
- 연차 정책 어디서 수정? → 연차 정책 관리 화면 > 행 편집 아이콘
- 연차 정책 어디서 삭제? → 연차 정책 관리 화면 > 행 삭제 아이콘
- 근속 N년 연차 미리보기 어디서? → 정책 등록 모달 > 근속별 연차 미리보기
