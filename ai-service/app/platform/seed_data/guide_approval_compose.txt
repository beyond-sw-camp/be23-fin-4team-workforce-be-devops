[결재] 결재 작성

■ 어떤 작업?
회사에 결재를 요청합니다. 모든 결재 양식(휴가/출장/사직서/공문 등)은 이 화면에서 작성합니다.

■ 어디서?
사이드바 → 전자결재 → 결재 생성

■ 작성 절차
1. 결재 화면 진입
2. "결재 생성" 클릭
3. 양식 선택 - 사용 가능한 양식 목록:
   - 휴가신청서 (연차/반차/경조/병가/예비군 등)
   - 연장근무신청 (사전/사후, 연장/야간/휴일)
   - 출퇴근시간 변경 신청서
   - 휴직 신청서 (출산/육아/장기병가 등)
   - 사직서
   - 근로계약서
   - 수당 변경 신청
   - 국내출장신청
   - 해외출장신청
   - 업무기안
   - 업무보고서
   - 공문
4. 입력 항목 작성
5. 결재 라인 확인
6. 제출 또는 임시저장

■ 자주 묻는 질문 - 결재 일반
- 결재 어떻게 작성해? → 전자결재 > 결재 생성
- 결재 어떻게 올려? → 전자결재 > 결재 생성
- 결재 어디서 시작해? → 전자결재 > 결재 생성
- 어떤 양식이 있어? → 휴가/출장/사직서/근로계약/공문 등 (회사별 다름)
- 임시저장 가능? → 가능, "내 기안 문서 > 임시 저장함"에서 확인
- 결재 양식 만들고 싶어 → 관리자만 가능 (전자결재 > 결재 관리자)
- 한 번 제출한 결재 수정하려면? → 회수 후 재작성
- 결재 처음 어떻게 시작해? → 전자결재 메뉴 > 결재 생성

■ 자주 묻는 질문 - 휴가 신청
- 휴가 어디서 신청? → 전자결재 > 결재 생성 > 휴가신청서
- 휴가 어떻게 신청해? → 전자결재 > 결재 생성 > 휴가신청서 양식 선택
- 연차 어디서 신청? → 전자결재 > 결재 생성 > 휴가신청서 (휴가 종류: 연차)
- 연차 어떻게 신청? → 휴가신청서에서 휴가 종류 "연차" 선택
- 반차 어디서 신청? → 전자결재 > 결재 생성 > 휴가신청서 (휴가 종류: 반차)
- 경조 휴가 신청은? → 휴가신청서에서 "경조" 선택
- 병가 어디서 신청? → 휴가신청서에서 "병가" 선택
- 출산 휴가 어디서? → 휴가신청서에서 "출산" 선택
- 예비군 휴가 신청? → 휴가신청서에서 "예비군" 선택
- 휴가 신청 누구에게 결재 가? → 결재 라인은 회사 정책에 따라 자동 지정
- 휴가 신청 취소하려면? → 결재 화면 > 내 기안 문서 > 해당 결재 회수
- 휴가 임시저장 가능? → 가능, 작성 중 임시저장 후 나중에 이어서 작성

■ 자주 묻는 질문 - 기타 결재
- 사직서 어디서 작성? → 전자결재 > 결재 생성 > 사직서
- 사직서 어떻게 올려? → 전자결재 > 결재 생성 > 사직서 양식 선택
- 출장 신청 어디서? → 전자결재 > 결재 생성 > 국내출장신청 또는 해외출장신청
- 휴직 신청 어디서? → 전자결재 > 결재 생성 > 휴직 신청서
- 공문 어디서 작성? → 전자결재 > 결재 생성 > 공문
- 업무기안 어디서? → 전자결재 > 결재 생성 > 업무기안

■ 주의 - 본인 휴가 현황 조회는 별도 화면
잔여 휴가, 사용한 휴가, 휴가 사용 비율 같은 본인 휴가 현황 조회는
"근태 관리 > 내 근태" 화면에서 확인합니다.
