[결재] 결재 대기 문서

■ 어떤 작업?
내가 결재해야 할 문서 목록을 확인하고 승인/반려 처리합니다.

■ 어디서?
사이드바 → 전자결재 → 결재 대기 문서

■ 화면 구성
- 내가 결재할 문서 목록
- 각 문서 상세 보기
- 승인/반려/보류 처리

■ 처리 절차
1. 결재 대기 문서 목록 확인
2. 클릭하여 상세 내용 검토
3. 의견 작성 (선택)
4. 승인 또는 반려 선택

■ 자주 묻는 질문
- 결재할 거 어디서 봐? → 전자결재 > 결재 대기 문서
- 결재 알림 왔는데 어디서 처리? → 결재 대기 문서 화면
- 결재 반려하려면? → 해당 문서 클릭 > "반려" 버튼
- 한 번 반려한 거 다시 승인하려면? → 신청자가 재작성/재상신해야 함
- 결재 끝난 거 보려면? → 결재 완료 문서 화면

■ 관련 키워드
결재 대기, 결재 처리, 승인, 반려, 결재할 문서,
결재 알림, 내가 결재, 결재 검토, 결재 사인,
결재 진행, 결재 처리

■ 관련 메뉴
관련 메뉴: /app/approvals
화면명: 결재 요청 작성 (결재 대기 문서 확인 및 처리)
