[인사관리] 휴가 종류 관리

■ 어떤 작업?
직원이 휴가 신청서에서 선택할 수 있는 휴가 종류를 관리합니다.
법정 기본 8종(연차, 반차(오전/오후), 경조휴가, 결혼, 출산, 부모 사망 등)은 회사 생성 시 자동 시드되며, 커스텀 휴가(예: 리프레시 휴가)도 추가할 수 있습니다.

■ 어디서?
사이드바 → 휴무 관리 → 휴가 종류 관리

■ 화면 구성
[목록 표]
- 이름 (휴가명) + "기본" 라벨 (시스템 기본인 경우)
- 잔고 유형: 당해 연차 / 차감 없음
- 유급: 유급 / 무급
- 연 한도: 연간 최대 사용 일수 (없으면 "—")
- 작업: 편집·삭제 아이콘 (시스템 기본 일부는 삭제 불가)

[우상단 버튼]
- "수동 휴가 부여": 특정 직원에게 휴가 일수를 직접 부여
- "기본 휴가 불러오기": 법정 기본 8종을 다시 시드 (소프트 삭제된 것 복원)
- "휴가 순서 수정": 휴가 신청서에 표시되는 순서 변경
- "휴가 종류 추가": 커스텀 휴가 신규 등록

■ 휴가 입력 항목 (휴가 종류 추가 시)
- 코드 (회사 내 고유, 예: REFRESH)
- 이름 (표시명, 예: 리프레시 휴가)
- 잔고 유형 (BalanceType): 연차 차감 여부
- 1회 사용 시 차감 일수 (연차=1.0, 반차=0.5)
- 유급/무급 여부
- 연간 최대 사용 일수 (제한 없으면 빈 값)
- 증빙 서류 필수 여부 (병가·경조 등)
- 표시 순서

■ 시스템 동작
- 시스템 기본(연차·반차 등)은 삭제 불가, 일부 항목만 수정 가능
- 커스텀 휴가는 전체 수정/삭제 가능
- (회사ID, 코드) UNIQUE: 같은 코드 중복 불가
- 차감 휴가는 잔고 차감, 미차감(경조 등)은 별도 처리

■ 자주 묻는 질문 (위치/방법 안내)
- 휴가 종류 어디서 추가? → 휴무 관리 > 휴가 종류 관리 > "휴가 종류 추가"
- 휴가 종류 어떻게 추가? → 휴가 종류 관리 화면 > 우상단 "휴가 종류 추가"

- 커스텀 휴가 어디서 만들어? → 휴무 관리 > 휴가 종류 관리 > "휴가 종류 추가"

- 경조 휴가 어디서 수정? → 휴가 종류 관리 > 해당 휴가 행 편집

- 직원에게 휴가 어디서 부여? → 휴무 관리 > 휴가 종류 관리 > "수동 휴가 부여"
- 휴가 수동 부여 어디서? → 휴가 종류 관리 > "수동 휴가 부여"
- 증빙 필수 휴가 어디서 설정? → 휴가 종류 관리 > 휴가 추가/수정 > 증빙 서류 필수 여부
- 유급 휴가 어디서 설정? → 휴가 종류 관리 > 휴가 추가/수정 > 유급 여부
