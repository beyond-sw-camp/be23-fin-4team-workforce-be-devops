[기타] 비밀번호 변경

■ 어떤 작업?
계정 비밀번호를 변경합니다.

■ 어디서?
사이드바 우측 상단 프로필 메뉴 → 비밀번호 변경
또는 로그인 후 별도 비밀번호 변경 화면으로 이동

■ 비밀번호 변경 시점
- 첫 로그인 시 비밀번호 강제 변경 화면으로 자동 이동
- 평소 보안을 위해 비밀번호를 주기적으로 바꿀 때
- 비밀번호 분실 시 (재설정과는 다름)

■ 비밀번호 분실/재설정의 경우
로그인 화면에서 "비밀번호 찾기" 클릭
이메일로 인증 코드 받아 재설정

■ 자주 묻는 질문
- 비밀번호 어디서 바꿔? → 비밀번호 변경 화면
- 비밀번호 잊어버렸어 → 로그인 화면 > 비밀번호 찾기
- 첫 로그인이라 비밀번호 바꾸라고 나옴 → 비밀번호 변경 화면 자동 이동, 그대로 변경 진행
- 비밀번호 자주 바꿔야 해? → 보안 정책에 따라 권장

■ 관련 키워드
비밀번호, 패스워드, 비번, 비밀번호 변경, 비밀번호 수정,
비밀번호 잊어버림, 비밀번호 분실, 비밀번호 찾기, 비밀번호 재설정,
패스워드 변경, 로그인 안 됨

■ 관련 메뉴
관련 메뉴: /change-password
