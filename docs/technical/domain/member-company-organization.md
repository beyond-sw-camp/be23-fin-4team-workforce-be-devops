# 회원·회사·조직 온보딩

## 개요

`member-service`는 회사 가입, 구성원, 조직, 직급/직책, 역할/권한, 인사 발령, 캘린더, 알림, ESG, 채팅 등 HRMS의 기본 사용자/조직 데이터를 담당합니다.

## 회사 온보딩

회사 가입 시 다음 초기 데이터가 함께 구성됩니다.

| 초기화 대상 | 설명 |
|-------------|------|
| 회사 정보 | 사업자 정보, 도메인, 관리자 계정 |
| 기본 조직 | 루트 조직 및 기본 부서 구조 |
| 직급/직책 | 기본 직급과 직책 |
| 역할/권한 | 시스템 관리자, HR 관리자 등 기본 역할 |
| 캘린더/공휴일 | 연도별 법정 공휴일 또는 회사 공휴일 |
| 급여/배치 | salary-service에 회사별 기본 배치 스케줄 초기화 요청 |

## 구성원 관리

- 구성원 등록 시 사번과 회사 이메일을 생성합니다.
- 임시 비밀번호를 발급하고 최초 로그인 시 비밀번호 변경 상태를 관리합니다.
- 구성원의 조직, 직급, 직책은 `MemberPosition` 중심으로 관리합니다.
- 인사 정보 변경 시 알림과 검색/RAG 연동 이벤트를 발행할 수 있습니다.

## 조직 관리

- 회사별 조직 트리를 관리합니다.
- 직급과 직책은 회사 단위로 독립 관리합니다.
- 조직 변경은 구성원, 평가, 결재선, 권한 범위에 영향을 주므로 회사 ID 기준으로 엄격히 분리합니다.

## 인사 발령

인사 발령은 결재 승인 이벤트 이후 실제 조직/직급/직책 변경과 이력 생성을 수행합니다. 승인 전에는 요청 데이터로 남기고, 승인 후 반영해 변경 근거를 추적합니다.
