[인사관리] 조직 관리 설정

■ 어떤 작업?
조직 구조, 직급, 직책, 역할·권한, 조직 개편(인사 발령)을 한 화면에서 통합 관리합니다.
실제 조직도 조회는 사이드바 좌하단 "전체 조직도"에서 합니다.

■ 어디서?
사이드바 → 인사 관리 → 조직 관리

■ 화면 구성 (5개 탭)

[조직 구조 탭]
- 회사 조직 트리 표시
- "수정하기" 버튼: 조직 생성/수정/삭제 편집 모드 진입
- "뷰 설정" 버튼: 트리 표시 옵션 변경

[직급 탭]
- 직급 목록 (직급명, 직급순서)
- 예: 관리자 / 임원 / 부장 / 차장 / 과장 / 대리 / 주임 / 사원
- "수정하기" 버튼: 직급 생성/수정/삭제

[직책 탭]
- 직책 목록 (직책명, 직책순서)
- 예: 관리자 / CEO / 본부장 / 실장 / 팀장 / 파트장 / 팀원
- "수정하기" 버튼: 직책 생성/수정/삭제

[역할·권한 탭]
- 역할 목록 (시스템 관리자 / 인사 관리자 / 팀장 / 일반 직원 등)
- "역할 추가" 버튼: 새 역할 생성
- 행별 편집·삭제 아이콘

[조직 개편(인사 발령) 탭]
- 좌측: 현재 조직도 (조회 전용)
- 우측: 시뮬 트리 (드래그로 직원 부서 이동, 클릭으로 직급 변경)
- "초기화" 버튼: 시뮬 트리 리셋
- "전자결재로 올리기" 버튼: 시뮬 결과를 인사 발령 결재로 상신

■ 자주 묻는 질문 (위치/방법 안내)
- 조직 어디서 만들어? → 인사 관리 > 조직 관리 > 조직 구조 탭 > "수정하기"
- 조직 어떻게 추가? → 조직 구조 탭에서 "수정하기" 클릭 후 트리에서 추가
- 부서 생성 어디서? → 인사 관리 > 조직 관리 > 조직 구조 탭
- 부서 어떻게 만들어? → 조직 구조 탭 > "수정하기" 모드에서 추가
- 부서 삭제 어디서? → 조직 관리 > 조직 구조 탭 > "수정하기" 모드
- 직급 어디서 추가? → 조직 관리 > 직급 탭 > "수정하기"
- 직급 어떻게 만들어? → 직급 탭에서 "수정하기" 클릭 후 추가
- 직급 순서 어디서 변경? → 조직 관리 > 직급 탭 > "수정하기"
- 직책 어디서 추가? → 조직 관리 > 직책 탭 > "수정하기"
- 직책 어떻게 만들어? → 직책 탭에서 "수정하기" 클릭 후 추가
- 역할 어디서 만들어? → 조직 관리 > 역할·권한 탭 > "역할 추가"
- 역할 어떻게 추가? → 역할·권한 탭 > "역할 추가" 버튼 클릭
- 권한 어디서 설정? → 조직 관리 > 역할·권한 탭
- 권한 어떻게 설정? → 역할·권한 탭에서 역할 편집 아이콘 클릭
- 역할 권한 변경 어디서? → 조직 관리 > 역할·권한 탭
- 조직 개편 어디서? → 인사 관리 > 조직 관리 > 조직 개편(인사 발령) 탭
- 인사 발령 어디서? → 조직 관리 > 조직 개편 탭
- 인사 발령 어떻게? → 조직 개편 탭에서 시뮬 트리 편집 후 "전자결재로 올리기"
- 부서 이동 시뮬 어디서? → 조직 관리 > 조직 개편 탭
- 직급 변경 시뮬 어디서? → 조직 관리 > 조직 개편 탭
