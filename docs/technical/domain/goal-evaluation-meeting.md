# 목표·평가·면담 도메인

## 개요

`goal-service`는 목표, 활동 기록, 목표 승인, 평가 시즌, 평가 설계, 평가 응답, 보정, 결과 발행, 면담 기록을 담당합니다.

## 목표 관리

| 기능 | 설명 |
|------|------|
| 목표 생성 | 개인/조직/전사 목표 생성 |
| KPI/OKR | 목표와 핵심 결과 관리 |
| 활동 기록 | 진행률 산정의 근거 데이터 |
| 목표 승인 | 목표 제출, 승인/반려 bundle 처리 |
| 댓글/반응 | 목표 단위 커뮤니케이션 |

## 평가 관리

| 기능 | 설명 |
|------|------|
| 시즌 | 평가 기간, 단계, 상태 관리 |
| 설계 | 평가 항목, 배점, 평가자 유형 |
| 그룹 | 평가 대상자 그룹 편성 |
| 평가자 배정 | 자기/상사/동료/하급자 평가자 매핑 |
| 응답 | 평가 답변 저장 |
| 보정 | 부서/그룹별 점수 보정과 최종 등급 |
| 결과 발행 | 평가 결과 확정 및 알림 |

## 스케줄러

- 평가 단계 전환
- 자기평가 마감 리마인드
- 결과 자동 발행
- 가중치 부족 리포트

## 면담

면담 기록과 액션 아이템을 관리합니다. 회의록 AI와 연결하면 회의에서 도출된 실행 항목을 목표 관리로 이어갈 수 있습니다.
