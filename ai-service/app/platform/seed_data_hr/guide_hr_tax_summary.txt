[인사관리] 세금·4대보험 집계

■ 어떤 작업?
선택한 월의 4대보험(국민연금·건강보험·고용보험·산재보험)과 원천세를 집계해 신고용 자료를 준비합니다.
직원 부담분은 실제 급여 명세서에서 공제된 값, 회사 부담분은 회사 부담률 기반 추정값으로 표시됩니다.

■ 어디서?
사이드바 → 급여 관리 → 세금·4대보험

■ 화면 구성
[월 선택 영역]
- "지난달" / "이번달" 빠른 선택 버튼
- 직접 월 선택 (예: 2026-05)
- 우상단 "신고용 엑셀" 다운로드 버튼

[안내]
- 참고용 집계 — 실제 신고는 외부 시스템 기준
- 회사 부담·산재보험은 단순 비율로 추정한 값
- 정확한 신고 금액은 4대사회보험 정보연계센터 또는 회계 시스템 사용 권장

[집계 표]
- 직원별: 국민연금 / 건강보험(+장기요양) / 고용보험 / 산재보험 / 소득세 / 지방소득세
- 직원 부담분: 실제 공제값 (확정된 명세서 기준)
- 회사 부담분: 회사 부담률 기반 추정값
- 합계 행

[빈 상태]
- 해당 월에 확정된 급여 데이터가 없으면 안내 메시지 표시
- 급여 정산 후 다시 확인 필요
