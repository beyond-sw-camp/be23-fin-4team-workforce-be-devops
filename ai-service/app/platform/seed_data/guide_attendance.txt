[근무] 내 근태

■ 어떤 작업?
본인의 출퇴근 기록, 근무 상태, 휴가 잔여일수를 한 화면에서 확인하고,
근태 정정/초과근무 신청을 결재로 올립니다.

■ 어디서?
사이드바 → 근태 관리 → 내 근태

■ 화면 구성
[일자별 탭]
근태 현황 영역:
- 오늘의 근무 상태 (출근 전/근무중/퇴근)
- 출근하기/퇴근하기 버튼
- 이번 달 근무일/지각/결근/조퇴 카운트

휴가 현황 영역:
- 연차휴가 (총 부여)
- 사용한 휴가 (승인 기준)
- 잔여 휴가 (사용 가능)
- 휴가 사용 비율
- "휴가 이력" 버튼

일자별 근태 표:
- 일자/상태/출근/퇴근/근무 시간/연장 시간/근태 정정/초과근무
- "근태정정신청" 버튼: 누락/오기록 정정 결재 작성
- "초과근무 신청" 버튼: 초과근무 결재 작성

[주간/월간 탭]
- 주·월 단위 근무시간 한도와 초과근무 흐름 확인
- 이번 주 근무시간 (40h 한도)
- 이번 주 연장근로 (12h 한도)
- 일별 근무 히트맵

■ 자주 묻는 질문 (위치/방법 안내 한정)
- 내 출퇴근 기록 어디서 봐? → 근태 관리 > 내 근태
- 출퇴근 기록 확인 어디서? → 근태 관리 > 내 근태
- 출근/퇴근 어디서 찍어? → 내 근태 화면 > 출근하기/퇴근하기 버튼
- 근태 어디서 봐? → 근태 관리 > 내 근태
- 출근 누락됐는데 어떻게 정정? → 내 근태 > 일자별 근태 > "근태정정신청" 버튼
- 근태 정정 신청 어디서? → 내 근태 > 일자별 근태 표 > "근태정정신청"
- 근태 정정 어떻게 해? → 내 근태 화면에서 해당 일자의 "근태정정신청" 버튼 클릭, 결재 작성으로 이동
- 초과근무 신청 어디서? → 내 근태 > 일자별 근태 표 > "초과근무 신청" 버튼
- 야근 신청 어디서? → 내 근태 화면 > "초과근무 신청" 버튼
- 연장근무 신청 어디서? → 내 근태 화면 > "초과근무 신청" 버튼
- 내 휴가 잔여일 어디서 봐? → 내 근태 화면 > 휴가 현황
- 잔여 연차 어디서 확인? → 내 근태 화면 > 휴가 현황 > 잔여 휴가
- 휴가 사용 이력 어디서 봐? → 내 근태 화면 > "휴가 이력" 버튼
- 휴가 며칠 썼는지 어디서 봐? → 내 근태 화면 > 휴가 현황
- 주간 근무시간 어디서 봐? → 내 근태 > 주간/월간 탭
- 월간 근무시간 어디서 봐? → 내 근태 > 주간/월간 탭
- 근무시간 한도 어디서 확인? → 내 근태 > 주간/월간 탭
