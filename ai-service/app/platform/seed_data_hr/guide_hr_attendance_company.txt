[인사관리] 전사 근태 현황

■ 어떤 작업?
회사 전체 직원의 근태(출퇴근/결근/휴가/외근/초과근무 등)를 일별·주별·월별로 한 화면에서 조회합니다.
초과근무 현황으로 이동하면 직원별 월 한도 대비 사용률까지 함께 확인할 수 있습니다.

■ 어디서?
사이드바 → 근태 관리 → 근태 현황

■ 화면 구성
[상단 - 기간 필터]
- 오늘 / 이번 주 / 이번 달 / 기간 선택 토글
- 날짜 선택기
- 이름·이메일·사번·부서 검색창
- "검색" 버튼
- "초과근무만" 토글: 초과근무한 직원만 필터

[요약 카드 6개]
- 정상 출근 / 결근 / 휴가·반차 / 출장·외근 / 평균 근무 / 초과근무 인원

[근태 표]
- 부서 / 사번 / 이름 / 일자 / 상태 / 첫 출근 / 마지막 퇴근 / 연장 / 근무

[우상단 버튼]
- "초과 근무 현황" 버튼 → 초과근무 현황 화면으로 이동

■ 초과 근무 현황 화면 (/app/attendance/overtime-status)
- 회사 월·일 연장근로 한도 표시 (예: 월 2,400분 = 40h, 일 600분 = 10h)
- 카드: 한도 초과(≥100%) / 임박(80~99%) / 관찰 대상 합계
- 집계 기간 선택 (주 단위 / 월 단위)
- 부서·직급·직책 필터, 사용률 토글(전체/≥50%/≥80%/≥100%)
- 직원별 표: 이번 달 총 근무시간 / 초과근무시간 / 승인된 연장근무 / 한도 대비 / 월 한도 초과분
- "근태 현황으로 이동" 버튼으로 복귀

■ 자주 묻는 질문 (위치/방법 안내)
- 전사 근태 어디서 봐? → 근태 관리 > 근태 현황
- 직원 출퇴근 어디서 확인? → 근태 관리 > 근태 현황
- 회사 전체 근태 어디서? → 근태 관리 > 근태 현황
- 결근자 어디서 확인? → 근태 현황 화면 > 결근 카드 또는 상태 필터
- 휴가 사용자 어디서 봐? → 근태 현황 > 상태 필터에서 휴가/반차
- 출장 외근자 어디서? → 근태 현황 > 상태 필터에서 출장/외근
- 초과근무 어디서 확인? → 근태 현황 > 우상단 "초과 근무 현황" 버튼
- 초과근무 현황 어디서 봐? → 근태 관리 > 근태 현황 > "초과 근무 현황"
- 직원별 초과근무 어디서? → 근태 현황 > "초과 근무 현황" 화면
- 한도 초과 직원 어디서 봐? → 초과 근무 현황 화면 > 한도 초과 카드
- 52시간 초과자 어디서? → 초과 근무 현황 화면 > 한도 초과(≥100%)
- 부서별 근태 어디서? → 근태 현황 화면 > 부서 검색
- 특정 직원 근태 어디서? → 근태 현황 > 이름·사번 검색
- 일별 근태 어디서 봐? → 근태 현황 > "오늘" 또는 날짜 선택
- 월별 근태 어디서? → 근태 현황 > "이번 달" 또는 기간 선택
