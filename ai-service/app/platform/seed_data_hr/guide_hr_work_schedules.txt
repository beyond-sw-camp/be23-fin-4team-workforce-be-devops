[인사관리] 근무 스케줄 관리

■ 어떤 작업?
회사 전체에 공통 적용되는 기본 근무 스케줄을 등록·수정합니다.
적용 기간, 근무 유형(고정/유연), 출퇴근 시각, 점심 시간을 설정하면 이후 일별 근태에 반영됩니다.

■ 어디서?
사이드바 → 근태 관리 → 근무스케줄 관리

■ 근무 유형
[고정 근무]
- 모든 직원에게 동일한 출근/퇴근 시각 적용
- 회사가 정한 점심 시간 일률 적용 (예: 12:00 ~ 13:00)
- 하루 규정 근무시간(분) 직접 설정
- "시차 출퇴근 시간대 관리" 버튼 비활성

[유연 근무 - 시차출퇴근제]
- 회사가 여러 출퇴근 슬롯을 제공
- 직원이 매월 슬롯 중 하나를 선택해 근무
- 고정 출퇴근 시각·점심시간 없음 (슬롯에 정의됨)
- 슬롯 선택 마감일 설정 (예: 25일까지 다음 달 슬롯 선택)
- "시차 출퇴근 시간대 관리" 버튼 활성 → 슬롯 등록·관리

■ 화면 구성
[목록 표]
- 이름 / 유형 / 대상 / 시간 / 근무(분) / 점심시간 / 적용기간 / 액션(수정·삭제)

[우상단 버튼]
- "스케줄 추가": 새 근무 스케줄 등록
- "시차 출퇴근 시간대 관리": 유연근무 스케줄의 슬롯 관리 화면으로 이동 (FLEXIBLE만)

■ 시차 출퇴근 시간대 관리 (/app/attendance/flexible-slots)
유연근무 스케줄에 속한 슬롯들을 등록·관리합니다.
- 근무 스케줄 선택 드롭다운
- 슬롯 표: 코드(자동) / 스케줄명(예: 9시) / 시간(09:00~18:00) / 점심시간 / 근무분 / 기본 / 액션
- "기본" 슬롯: 신규 입사자에게 자동 할당, 미선택자도 자동 적용
- "스케줄 등록" 버튼: 새 슬롯 추가

■ 자주 묻는 질문 (위치/방법 안내)
- 근무 스케줄 어디서 만들어? → 근태 관리 > 근무스케줄 관리 > "스케줄 추가"
- 근무 스케줄 어떻게 추가? → 근무스케줄 관리 화면 > "스케줄 추가" 버튼
- 회사 출퇴근 시간 어디서 설정? → 근태 관리 > 근무스케줄 관리
- 점심 시간 어디서 설정? → 근무스케줄 관리 > 스케줄 추가/수정
- 근무 스케줄 어디서 수정? → 근무스케줄 관리 화면 > 행 액션 편집 아이콘
- 근무 스케줄 어디서 삭제? → 근무스케줄 관리 화면 > 행 액션 삭제 아이콘
- 고정 근무제 어디서 설정? → 근무스케줄 관리 > "스케줄 추가" > 유형 "고정"
- 유연근무제 어디서 설정? → 근무스케줄 관리 > "스케줄 추가" > 유형 "유연"
- 시차 출퇴근 어디서? → 근무스케줄 관리 > 우상단 "시차 출퇴근 시간대 관리"
- 시차 출퇴근 시간대 관리 어디서? → 근태 관리 > 근무스케줄 관리 > "시차 출퇴근 시간대 관리"
- 시차 슬롯 어디서 추가? → 시차 출퇴근 시간대 관리 화면 > "스케줄 등록"
- 출퇴근 슬롯 어디서 만들어? → 근무스케줄 관리 > "시차 출퇴근 시간대 관리" > "스케줄 등록"
- 9시 출근 슬롯 어디서 추가? → 시차 출퇴근 시간대 관리 > "스케줄 등록"
- 기본 슬롯 어디서 지정? → 시차 출퇴근 시간대 관리 > 행 액션 "기본" 토글
- 슬롯 선택 마감일 어디서? → 근무스케줄 관리 > 스케줄 추가/수정 > 선택 마감일 입력
