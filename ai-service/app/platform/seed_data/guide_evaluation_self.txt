자기평가 작성 가이드

[어디서 작성하나요]
관련 메뉴: /app/evaluations
화면명: 내 평가
탭: 평가 작성

[자기평가가 무엇인가요]
시즌 활성화 시 본인의 승인된 개인 목표가 봉인되어 평가 대상이 됩니다.
각 목표별로 본인이 직접 평가합니다.

[작성 가능 시점]
- 시즌 단계가 SELF_EVAL일 때
- 본인 응답 단계가 SELF_PENDING이어야 입력 가능
- 한 번 제출 후에는 수정 불가 (확인만 가능)

[작성 화면 구성]
- 상단: 평가 요약 카드 (예상 점수 등)
- 중간: 개인 목표별 카드 (목표 수만큼)
- 하단: 종합 코멘트 카드

[목표별 카드 입력 항목]
각 개인 목표 카드마다:

1. 등급 (S/A/B/C) - 필수 (제출 시)
   - 라디오 버튼 선택
   - 조직 목표의 등급 기준을 참고하여 본인 등급 선택
   - 조직 목표 등급 기준이 카드에 표시됨

2. 개인 목표 코멘트 - 선택
   - 짧은 한 줄 입력
   - 목표 달성에 대한 본인 설명

3. 증빙 - 선택
   - 파일 업로드 또는 URL 입력
   - 등급 선택을 뒷받침하는 자료

[종합 코멘트]
시즌 전체에 대한 본인 코멘트 (선택)

[점수는 어떻게]
점수는 직접 입력하지 않습니다. 선택한 등급과 가중치로 자동 계산됩니다.

기본 환산 (회사 설정에 따라 다를 수 있음):
- S = 100점
- A = 85점
- B = 70점
- C = 55점

최종 점수 = Σ (목표별 점수 × 가중치) / 100

[임시 저장 vs 제출]
- 임시 저장: 등급 일부만 입력해도 가능. 나중에 이어서 작성
- 제출: 모든 목표에 등급 선택 필수. 제출 후 수정 불가

[자주 묻는 질문]
Q: 등급 안 고른 목표가 있어도 임시 저장 가능?
A: 가능합니다. 다만 제출은 모든 목표 등급 입력 시에만.

Q: 증빙 안 첨부해도 되나요?
A: 선택 사항입니다. 다만 등급의 설득력을 위해 첨부 권장.

Q: 제출 후 수정하고 싶어요.
A: 수정 불가합니다. 단계가 SELF_PENDING이 아니면 입력 비활성화.
