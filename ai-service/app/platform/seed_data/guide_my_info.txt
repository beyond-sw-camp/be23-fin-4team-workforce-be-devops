[기타] 내 정보 수정

■ 어떤 작업?
직원 본인의 개인 정보 (연락처, 주소, 계좌번호 등)를 수정합니다.

■ 어디서?
사이드바 우측 상단 프로필 메뉴 → 내 정보
"내 정보 수정" 버튼 클릭하여 편집 모드 진입

■ 수정 가능한 항목
연락처 섹션:
- 휴대폰 번호
- 연락처 공개 여부 (공개/본인만)
- 내선번호
- 직통번호
- 비상연락처

주소 섹션:
- 주소
- 상세 주소
- 주소 공개 여부 (공개/본인만)

급여 계좌 섹션:
- 은행
- 계좌번호

기타:
- 프로필 사진 (업로드/변경/삭제)

■ 자주 묻는 질문
- 내 정보 어디서 수정? → 프로필 메뉴 > 내 정보 > "내 정보 수정" 버튼
- 내 정보 어떻게 수정? → 프로필 메뉴 > 내 정보 진입 후 "내 정보 수정" 버튼 클릭
- 계좌번호 어디서 수정? → 내 정보 화면 > "내 정보 수정" 버튼 > 급여 계좌 섹션
- 계좌번호 어떻게 변경? → 내 정보 화면 > "내 정보 수정" 버튼 > 급여 계좌 섹션
- 은행 변경하려면? → 내 정보 화면 > "내 정보 수정" 버튼 > 급여 계좌 섹션
- 휴대폰 번호 바꾸려면? → 내 정보 화면 > "내 정보 수정" 버튼 > 연락처 섹션
- 주소 변경하려면? → 내 정보 화면 > "내 정보 수정" 버튼 > 주소 섹션
- 프로필 사진 바꾸려면? → 내 정보 화면 > "내 정보 수정" 버튼
- 비상연락처 등록 어디서? → 내 정보 화면 > "내 정보 수정" 버튼 > 연락처 섹션
- 내선번호 등록은? → 내 정보 화면 > "내 정보 수정" 버튼 > 연락처 섹션
- 이름이나 이메일 수정은? → 인사팀에 요청해야 합니다 (개인 수정 불가)
