[인사관리] 결재 관리자 (양식·정책라인·전자계약·인감)

■ 어떤 작업?
회사에서 사용하는 결재 양식, 양식별 결재 라인, 전자계약 양식, 인감을 통합 관리합니다.
인사팀(또는 결재 관리 권한자)이 사용하는 관리 화면입니다.

■ 어디서?
사이드바 → 인사 관리 → 결재 양식 설정

■ 화면 구성 (4개 탭)

[결재 양식 관리 탭]
- 회사에서 사용 중인 결재 양식 목록
  예: 업무보고서, 공문, 연장근무신청, 업무기안, 휴가신청서, 출퇴근시간 변경 신청서, 사직서, 휴직 신청서 등
- 각 양식의 요청 유형(일반기안/공문/근태/휴가/인사 등) 표시
- 활성/비활성 토글로 양식 사용 여부 제어
- 편집 아이콘으로 양식 수정
- "정책라인 확인" 링크로 결재 라인 확인
- "양식 추가" 버튼: 새 결재 양식 생성
- "새로고침" 버튼

[정책라인 관리 탭]
- 양식별 결재 순서(직책/단계)를 설정
- "양식 선택" 드롭다운으로 양식 선택 후 정책라인 편집
- 결재선 N개 (순서/직책/조직(선택)/관리)
- "수정하기" 버튼: 정책라인 편집 모드 진입
- 저장 시 기존 정책라인은 전체 교체됨

[전자계약 양식 관리 탭]
- 전자계약 템플릿 목록 (연봉계약서, 개인정보 수집·이용 동의서, 근로계약서, 비밀유지서약서 등)
- 계약 유형 / 사용 상태(활성/비활성) / 수정일 표시
- 발송 화면에는 활성 템플릿만 노출됨
- "새 계약서 양식" 버튼: 새 템플릿 추가
- 편집 아이콘으로 양식 수정

[인감 관리 탭]
- 결재 및 계약 문서에 사용할 공식 인감 등록·관리
- 인감 이미지 업로드 (투명 배경 PNG/JPG 권장, PDF도 지원)
- 최대 5MB, 저장 시 기존 인감 교체
- 미리보기 영역
- "파일 선택" / "삭제" / "저장" 버튼

■ 자주 묻는 질문 (위치/방법 안내)
- 결재 양식 어디서 만들어? → 인사 관리 > 결재 양식 설정 > 결재 양식 관리 탭 > "양식 추가"
- 결재 양식 어떻게 추가? → 결재 양식 관리 탭 > "양식 추가" 버튼 클릭
- 결재 양식 어디서 수정? → 결재 양식 관리 탭 > 해당 양식 행의 편집 아이콘
- 결재 양식 비활성화 어디서? → 결재 양식 관리 탭 > 사용 상태 토글
- 결재 라인 어디서 설정? → 인사 관리 > 결재 양식 설정 > 정책라인 관리 탭
- 결재 라인 어떻게 만들어? → 정책라인 관리 탭 > 양식 선택 후 "수정하기"
- 결재 순서 어디서 변경? → 결재 양식 설정 > 정책라인 관리 탭
- 결재 단계 어디서 추가? → 정책라인 관리 탭 > 양식 선택 > "수정하기"
- 정책라인 어디서? → 인사 관리 > 결재 양식 설정 > 정책라인 관리 탭
- 정책라인 어떻게 설정? → 정책라인 관리 탭 > 양식 선택 > "수정하기" > 결재선 추가
- 전자계약 양식 어디서 추가? → 결재 양식 설정 > 전자계약 양식 관리 탭 > "새 계약서 양식"
- 전자계약 템플릿 어디서 만들어? → 전자계약 양식 관리 탭 > "새 계약서 양식" 버튼
- 근로계약서 양식 어디서 수정? → 전자계약 양식 관리 탭 > 해당 양식 편집 아이콘
- 인감 어디서 등록? → 인사 관리 > 결재 양식 설정 > 인감 관리 탭
- 인감 어떻게 업로드? → 인감 관리 탭 > "파일 선택" 후 "저장"
- 회사 인감 어디서 관리? → 결재 양식 설정 > 인감 관리 탭
- 인감 이미지 변경 어디서? → 인감 관리 탭 > "파일 선택"으로 새 이미지 업로드 후 "저장"
