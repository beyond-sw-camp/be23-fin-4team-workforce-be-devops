# 근태·휴가 도메인

## 개요

`salary-service`의 `attendance` 패키지는 출퇴근, 일별/월별 근태, 휴가 잔고, 연장근로, 외근/출장, 시차출퇴근, 연차 촉진을 담당합니다.

## 주요 기능

| 기능 | 설명 |
|------|------|
| 출퇴근 로그 | 출근/퇴근 기록, 취소, 일별 집계 |
| 일근태/월근태 | 휴가, 외근, 연장근무를 반영한 근태 장부 |
| 휴가 잔고 | 부여, 차감, 복구, 이월 동의 |
| 휴가 신청 | 결재 승인 후 잔고 차감과 근태 반영 |
| 연차 촉진 | 1차/2차 알림, 미응답자 조회, 강제 지정 |
| 연장근무 | 정책 검증, 사전/사후 신청, 승인 반영 |
| 시차출퇴근 | 슬롯 선택, 마감일 이후 자동 배정 |
| 출장/외근 | 신청/승인/근태 반영 |

## 결재 이벤트 연동

근태·휴가 도메인은 approval-service의 이벤트를 구독합니다.

| 이벤트 | 처리 |
|--------|------|
| LeaveApprovalEvent | 휴가 승인/반려에 따른 잔고 및 근태 반영 |
| OvertimeApprovalEvent | 연장근무 승인 상태 반영 |
| ScheduleApprovalEvent | 근무 스케줄 변경 반영 |
| AttendanceCorrectionApprovalEvent | 출퇴근 정정 반영 |
| LeaveOfAbsenceApprovalEvent | 휴직 반영 |

## 정책 검증

- 주 52시간 한도
- 일/월 연장근로 한도
- 잔여 연차 부족 여부
- 휴가 정책의 부여/이월/정산 기준
- 사후 초과근무 신청 가능 시간
