[인사관리] 구성원 관리

■ 어떤 작업?
조직 구성원을 검색·조회하고, 직원 계정을 생성하며, 인사 정보를 수정합니다.
직원의 휴직/복직 처리, 잠금 해제도 이 화면에서 진행합니다.

■ 어디서?
사이드바 → 인사 관리 → 구성원

■ 화면 구성
- 직원 검색창 (이름·이메일·부서)
- 직원 목록 표 (이름/이메일/부서/상태)
- 우상단 "직원 계정 생성" 버튼
- 행 클릭 시 직원 상세 화면으로 이동

■ 직원 상세 화면 기능
- "인사 정보 수정" 버튼: 이름, 사번, 입사일, 고용형태, 재직 상태, 조직, 직급, 직책, 역할 변경
- 변경 시 변경 유형, 변경 사유 입력 필수
- "직원 이력 보기" 버튼: 조직 이동, 직급 변경, 고용 상태 변경 이력 조회
- "휴직 처리" / "복직 처리" 버튼
- "잠금 해제" 버튼 (직원이 비밀번호 5회 이상 틀려 잠긴 경우)
- "급여 등록" 버튼 (기본급 미등록 시 표시됨)

■ 자주 묻는 질문 (위치/방법 안내)
- 직원 어디서 등록해? → 인사 관리 > 구성원 > 우상단 "직원 계정 생성" 버튼
- 직원 어떻게 등록? → 구성원 화면에서 "직원 계정 생성" 버튼 클릭
- 직원 계정 생성 어디서? → 인사 관리 > 구성원 > "직원 계정 생성"
- 신규 직원 추가 어디서? → 인사 관리 > 구성원 > "직원 계정 생성"
- 직원 정보 어디서 수정? → 구성원 > 해당 직원 클릭 > "인사 정보 수정"
- 직원 정보 어떻게 수정? → 구성원 화면 > 직원 클릭 후 "인사 정보 수정" 버튼
- 직원 이력 어디서 봐? → 구성원 > 직원 상세 > "직원 이력 보기" 버튼
- 조직 이동 이력 어디서? → 구성원 > 직원 상세 > "직원 이력 보기"
- 직급 변경 이력 어디서? → 구성원 > 직원 상세 > "직원 이력 보기"
- 고용 상태 이력 어디서? → 구성원 > 직원 상세 > "직원 이력 보기"
- 휴직 처리 어디서? → 구성원 > 직원 상세 > "휴직 처리" 버튼
- 휴직 어떻게 처리? → 직원 상세 화면에서 "휴직 처리" 클릭
- 복직 처리 어디서? → 구성원 > 직원 상세 > "복직 처리" 버튼
- 복직 어떻게 처리? → 직원 상세 화면에서 "복직 처리" 클릭
- 직원 잠금 어디서 풀어? → 구성원 > 직원 상세 > "잠금 해제" 버튼
- 비밀번호 잠금 어디서 해제? → 구성원 > 직원 상세 > "잠금 해제"
- 잠금 해제 어떻게? → 직원 상세 화면 > "잠금 해제" 버튼 클릭
- 직원 검색 어떻게? → 구성원 화면 상단 검색창에 이름·이메일·부서 입력
- 재직자 목록 어디서 봐? → 인사 관리 > 구성원
