[인사관리] 급여 정산 관리

■ 어떤 작업?
매월 직원 급여를 정산·확정·지급하는 핵심 화면입니다.
정기 급여, 상여금, 퇴직 정산, 수당 부여를 한 곳에서 처리하고 급여 변동 이력까지 관리합니다.

■ 어디서?
사이드바 → 급여 관리 → 급여 정산 관리

■ 화면 구성 (7개 탭)

[정산 처리 탭]
이번 달 급여 명세서를 생성하고 검토·확정·지급까지 처리하는 메인 화면.
- 안내 카드: 이번 달 정기급여 명세서 생성 여부, 정산일 표시
- 처리 대상 / 검토 전 / 지급 대기 카드
- 합계 카드: 정기급여 / 퇴직정산 / 상여성과 / 총지급 / 총공제 / 실수령 합계
- 검색·필터: 이름·사번·부서, 상태, 급여구분, 부서
- 직원별 행: 사번 / 이름 / 부서 / 급여 종류 / 정산 대상 / 상태 / 총지급 / 총공제 / 실수령
- "일괄 확정" / "일괄 지급" / "일괄 삭제" 버튼 (선택 건수)
- 우상단: "엑셀 다운로드" / "명세서 생성" 버튼

[지급 이력 탭]
지난달까지 확정·지급된 급여 명세서 이력 조회.
- 월 선택 (예: 2026-04)
- 합계: 정기급여 / 퇴직정산 / 상여성과 / 총지급 / 총공제 / 실수령
- 직원별 행: 사번 / 이름 / 부서 / 정산 대상 / 급여구분 / 총지급 / 총공제 / 실수령 / 지급일

[급여 등록 탭]
신규 입사자나 급여가 누락된 재직자에게 기본급·호봉을 등록.
- 전체 재직자 / 급여 미등록자 카운트
- 직원별 행: 사번 / 이름 / 부서 / 계좌(등록·미등록) / 액션 "급여 등록"
- 안내: 신규 입사자는 임시로 0원(연봉제) / 1호봉(호봉제)으로 자동 생성됨, 정확한 금액은 급여 등록으로 수정 필요

[상여금 지급 탭]
현재 활성 상여금 정책 기반으로 정기상여·성과급·명절상여를 일회성 지급.
- 현재 활성 상여금 정책 안내
- 입력: 상여금 유형 (정기상여/명절상여/성과급) / 지급일 / 지급 비율(%) / 메모
- "시뮬 미리보기" 버튼으로 대상자별 예상 지급액 확인 후 확정

[퇴직 정산 탭]
사직서 결재 승인된 직원의 퇴직 정산을 처리.
- 검토 전 / 지급 대기 / 지급 완료 / 누적 지급액 카드
- 안내: 사직 결재 승인 시 회사 퇴직급여 정책(법정/DB/DC)에 따라 자동 생성
- 안내: 1년 미만 근속은 0원, 실수령액 = 퇴직금 + 미사용 연차 수당 + 퇴직월 일할 급여 − 공제
- 직원별 행: 직원 / 퇴직일 / 실수령액 / 총지급 / 상태 / 지급일 / 생성일 / 상세
- "일괄 확정" / "일괄 지급" 버튼

[급여 변동 이력 탭]
직원별 연봉·호봉 변경 이력을 등록·관리.
- 검색·필터: 이름·사번·부서, 상태, 부서, 신규 입사자(급여 등록 대상)만
- 직원별 행: 사번 / 이름 / 부서 / 호봉 / 기본급 / 직급 / 직책 / 부양가족 / 적용 기간
- "급여 등록" 버튼: 연봉 인상·직급 변경 시 새 이력 추가

[수당 관리 탭]
자격수당, 직책수당 등 개인 차등 수당을 즉시 부여하고 적용 현황 조회.
- 대상 직원 / 지급 완료·중 / 총 지급 / 총 공제 / 실수령 합계 카드
- "일괄 수당 부여" 버튼
- 항목·부서·이름 필터, 검색
- 태그 색상: 청록(회사 공통) / 파랑(개인 차등)
- 안내: 공제·실수령은 과세 수당 기준 추정치(약 15%), 실제 값은 정산 시 확정

■ 자주 묻는 질문 (위치/방법 안내)
- 급여 정산 어디서 해? → 급여 관리 > 급여 정산 관리 > 정산 처리 탭
- 이번 달 급여 명세서 어디서 만들어? → 급여 정산 관리 > 정산 처리 탭 > 우상단 "명세서 생성"
- 명세서 어떻게 만들어? → 정산 처리 탭에서 "명세서 생성" 버튼 클릭
- 급여 일괄 확정 어디서? → 급여 정산 관리 > 정산 처리 탭 > 직원 선택 후 "일괄 확정"
- 급여 일괄 지급 어디서? → 급여 정산 관리 > 정산 처리 탭 > "일괄 지급"
- 급여 명세서 엑셀 어디서? → 급여 정산 관리 > 정산 처리 탭 > 우상단 "엑셀 다운로드"
- 지난 달 급여 어디서 봐? → 급여 정산 관리 > 지급 이력 탭
- 급여 지급 이력 어디서? → 급여 관리 > 급여 정산 관리 > 지급 이력 탭
- 신규 입사자 급여 어디서 등록? → 급여 정산 관리 > 급여 등록 탭
- 직원 기본급 어디서 등록? → 급여 정산 관리 > 급여 등록 탭 > "급여 등록"
- 누락된 직원 급여 어디서? → 급여 정산 관리 > 급여 등록 탭 (미등록 표시 직원)
- 상여금 어디서 지급? → 급여 정산 관리 > 상여금 지급 탭
- 명절 상여 어디서 줘? → 급여 정산 관리 > 상여금 지급 탭 > 상여금 유형 "명절상여"
- 정기 상여 어디서 지급? → 급여 정산 관리 > 상여금 지급 탭 > "정기상여"
- 성과급 어디서 지급? → 급여 정산 관리 > 상여금 지급 탭 > "성과급"
- 퇴직 정산 어디서? → 급여 관리 > 급여 정산 관리 > 퇴직 정산 탭
- 퇴직금 어디서 처리? → 급여 정산 관리 > 퇴직 정산 탭
- 퇴직금 일괄 지급 어디서? → 급여 정산 관리 > 퇴직 정산 탭 > "일괄 지급"
- 연봉 인상 어디서 등록? → 급여 정산 관리 > 급여 변동 이력 탭 > "급여 등록"
- 연봉 변경 어디서? → 급여 관리 > 급여 정산 관리 > 급여 변동 이력 탭
- 호봉 변경 어디서 등록? → 급여 정산 관리 > 급여 변동 이력 탭 > "급여 등록"
- 직급 변경 어디서 반영? → 급여 정산 관리 > 급여 변동 이력 탭에서 새 이력 등록
- 자격 수당 어디서 부여? → 급여 정산 관리 > 수당 관리 탭 > "일괄 수당 부여"
- 직책 수당 어디서 줘? → 급여 정산 관리 > 수당 관리 탭
- 수당 일괄 부여 어디서? → 급여 정산 관리 > 수당 관리 탭 > "일괄 수당 부여"
- 직원별 수당 현황 어디서 봐? → 급여 정산 관리 > 수당 관리 탭
