목표 승인 흐름 가이드

[승인이 필요한 이유]
개인 목표는 팀장 승인을 받아야 평가 대상이 됩니다.
승인된 목표만 평가 시즌 활성화 시 자동 로드되어 자기평가/상사평가 대상이 됩니다.

[팀원 입장 - 승인 요청 제출]

어디서 제출:
관련 메뉴: /app/performance?view=my
화면명: 성과 관리 > 내 목표

제출 조건:
- 본인 개인 목표 모두 작성 완료
- 가중치 합이 정확히 100%
- 같은 사이클(예: 2026 1분기) 단위로 묶어서 제출

제출 절차:
1. 내 목표 화면 하단 "승인 요청" 버튼 클릭
2. 팀장에게 승인 요청 전송
3. 제출 후 승인 대기 상태로 전환

상태 확인:
- 내 목표 화면에서 승인 상태 확인
- 상태값: 승인 대기 / 승인됨 / 반려됨

[팀장 입장 - 승인 처리]

어디서 처리:
관련 메뉴: /app/performance?view=org
화면명: 성과 관리 > 조직 목표 관리
승인 큐: 같은 화면 내 "팀 목표 승인 큐"

처리 절차:
1. 팀 목표 승인 큐에서 팀원의 목표 묶음 선택
2. 목표별 내용 검토 (제목, 설명, 가중치, 정렬된 조직 목표)
3. 승인 또는 반려
4. 반려 시 사유 작성 → 팀원이 수정 후 재제출

[승인 후 흐름]
1. 팀원 개인 목표 확정
2. 평가 시즌 활성화 시 자동 로드
3. goalSnapshot 봉인 → 변경 불가
4. 자기평가 단계에서 목표별 평가 진행
