[휴가] 휴가 신청

■ 어떤 작업?
연차/반차/경조사/병가 등 휴가를 결재로 신청합니다.

■ 어디서?
사이드바 → 전자결재 → 결재 생성 → "휴가신청서" 양식 선택

■ 신청 절차
1. 결재 화면 진입
2. "결재 생성" 클릭
3. 양식 목록에서 "휴가신청서" 선택
4. 입력 항목 작성:
   - 제목 (필수)
   - 휴가 종류 선택 (필수): 연차/반차/경조/병가/예비군 등
   - 시작일 (필수)
   - 종료일 (필수)
   - 사용일수 (필수)
   - 휴가 사유 (선택)
5. 결재 라인 확인
6. 제출

■ 자주 묻는 질문
- 휴가 어떻게 신청해? → 전자결재 > 결재 생성 > 휴가신청서
- 연차 신청 어디서? → 전자결재에서 "휴가신청서" 양식
- 반차 신청도 같은 양식? → 네, "휴가 종류"에서 반차 선택
- 경조 휴가 신청은? → 휴가신청서에서 "경조" 선택
- 휴가 신청하면 누구에게 결재 가? → 결재 라인은 회사 정책에 따라 자동 지정
- 휴가 신청 취소하려면? → 결재 화면 > 내 기안 문서 > 해당 결재 회수
- 임시저장 가능? → 가능, 작성 중 임시저장 후 나중에 이어서 작성
