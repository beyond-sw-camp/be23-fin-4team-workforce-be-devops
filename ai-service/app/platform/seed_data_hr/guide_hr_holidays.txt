[인사관리] 회사 공휴일 관리

■ 어떤 작업?
회사가 인정하는 휴일(법정 공휴일 + 회사 지정 휴일)을 등록·수정·삭제합니다.
법정 공휴일은 API로 일괄 가져올 수 있고, 창립기념일 같은 회사 자체 휴일은 직접 추가합니다.
유급/무급 구분에 따라 급여 계산에 자동 반영됩니다.

■ 어디서?
사이드바 → 휴무 관리 → 공휴일 관리

■ 화면 구성
[법정 공휴일 불러오기 영역]
- 대상 연도 입력 (예: 2026)
- "법정 공휴일 새로고침" 버튼: 외부 API 연동으로 해당 연도의 법정 공휴일 일괄 등록
- 직접 등록한 회사 휴일은 새로고침해도 유지됨

[월별 캘린더]
- 좌우 화살표로 월 이동
- "오늘" 버튼
- 날짜 셀에 등록된 공휴일 이름 표시 (예: 노동절, 어린이날)
- 날짜 클릭 시 우측 패널에서 해당 날짜 공휴일 확인·관리

[우측 패널 - 선택한 날짜]
- 선택한 날짜 표시
- "이 날짜에 공휴일 추가" 버튼
- 등록된 공휴일 목록 (없으면 "이 날짜에 등록된 공휴일이 없습니다.")

[우상단]
- "공휴일 추가" 버튼: 임의 날짜에 공휴일 등록

■ 공휴일 입력 항목
- 공휴일 이름 (예: 설날, 창립기념일)
- 날짜
- 유급 여부 (유급 / 무급)
- 법정 공휴일 여부 (자동 표시)

■ 시스템 동작
- (회사ID, 날짜) UNIQUE 제약: 같은 날 중복 등록 불가
- 유급 휴일은 급여 계산 시 정상 근무로 처리
- 월간 근태표에서 공휴일이 시각적으로 표시됨

■ 자주 묻는 질문 (위치/방법 안내)
- 공휴일 어디서 등록? → 휴무 관리 > 공휴일 관리 > 우상단 "공휴일 추가"
- 공휴일 어떻게 추가? → 공휴일 관리 화면 > "공휴일 추가" 또는 캘린더에서 날짜 선택 후 "이 날짜에 공휴일 추가"
- 법정 공휴일 어디서 가져와? → 휴무 관리 > 공휴일 관리 > 연도 입력 후 "법정 공휴일 새로고침"
- 법정 공휴일 어떻게 불러와? → 공휴일 관리 화면 > 대상 연도 입력 > "법정 공휴일 새로고침"
- 2026년 공휴일 어디서 가져와? → 공휴일 관리 > 연도 2026 입력 > "법정 공휴일 새로고침"
- 창립기념일 어디서 등록? → 휴무 관리 > 공휴일 관리 > "공휴일 추가"
- 회사 휴일 어디서 추가? → 휴무 관리 > 공휴일 관리
- 임시공휴일 어디서? → 공휴일 관리 > "공휴일 추가"
- 유급 휴일 어디서 설정? → 공휴일 관리 > 공휴일 추가/수정 시 유급 여부 선택
- 무급 휴일 어디서 설정? → 공휴일 관리 > 공휴일 추가/수정 시 유급 여부 "무급" 선택
- 공휴일 어디서 수정? → 공휴일 관리 > 캘린더에서 날짜 선택 > 우측 패널에서 편집
- 공휴일 어디서 삭제? → 공휴일 관리 > 캘린더에서 날짜 선택 > 우측 패널에서 삭제
- 등록한 공휴일 어디서 봐? → 휴무 관리 > 공휴일 관리 (캘린더에 표시)
